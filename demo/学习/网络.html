URI: Uniform Resource Identifier
         即统一资源标识符，用来唯一的标识一个资源

    RUL: Uniform Resource Locator
         统一资源定位符。URL可以用来标识一个资源，而且还指明了如何定位这个资源。用地址定义一个资源。
      
    URN: Uniform Resource name
         统一资源命名。即通过名字来标识资源的。用名称定位一个资源

    URL与URN是URI的子集

    1. URL肯定是一个URI，URI并不一定是URL，也可能是URN。
       https 默认端口：443  http默认端口：80  
       URI 只是资源标识
       URL： 1、资源标识；
             2、具有定位资源的功能
             3、指明了获取资源所采用的协议
             协议名称 + 主机名称 + 端口号 + 路径 + 文件 + 查询所需字符串
             http:// jsplusplus.com:80  index/index.html  ?a=1&b=2#d
        URN：jsplusplus.com:80/index/index.html?a=1&b=2#d
